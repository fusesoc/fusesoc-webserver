FuseSoC Package Directory documentation
=======================================

FuseSoC Package Directory is an open source package manager for hardware IP cores
It provides a web interface and API for browsing, searching, and publishing FuseSoC-compatible cores.

FuseSoC is an open source package manager for hardware IP cores. See the `FuseSoC GitHub repository <https://github.com/olofk/fusesoc>`_ for more information.
